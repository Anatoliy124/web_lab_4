# Приложение для прогноза погоды

Веб-приложение для просмотра прогноза погоды с поддержкой геолокации и добавления нескольких городов.

## Функциональность

- Запрос геолокации при первом открытии
- Отображение прогноза погоды на сегодня и минимум на 2 следующих дня
- Форма добавления города при отклонении геолокации
- Добавление нескольких городов (минимум 2 дополнительных)
- Автодополнение при вводе названия города
- Кнопка обновления данных
- Сохранение состояния в localStorage
- Адаптивный дизайн для мобильных и десктопных устройств

## Запуск

```bash
python3 -m http.server 8000
```

Откройте в браузере: `http://localhost:8000`

## Используемые API

**Open-Meteo API** - получение данных о погоде (бесплатно, без ключа)
- URL: https://api.open-meteo.com

**Nominatim API** - геокодирование (бесплатно, без ключа)
- URL: https://nominatim.openstreetmap.org

## Технологии

- HTML5, CSS3, Vanilla JavaScript
- Open-Meteo API, Nominatim API
- localStorage для сохранения состояния
